Ono 99 Calculator



An interactive Android calculator inspired by the ONO 99 card game, designed to track numbers with history and smart controls.



&nbsp;Description



Ono 99 Calculator helps players track the current number during ONO 99 games.

It includes operation history, undo support, and automatic button blocking.



Built to be fast, simple, and comfortable.



&nbsp;Features (Version 1.0)



&nbsp;Quick add buttons: +1 to +10



&nbsp;Quick subtract button: -10



&nbsp;Back button to return to previous value



&nbsp;Reset button clears value and history



&nbsp;Dropdown history menu (three-lines icon)



&nbsp;Each history item stores:



Operation text



Final result



&nbsp;Tap any history item to restore that state



&nbsp;Haptic vibration feedback



&nbsp;Automatic button blocking to avoid passing 99



&nbsp;Clean, centered UI



&nbsp;How it works



Each operation is stored inside a history list with:



Human-readable operation text



Final numeric value



You can navigate backward using Back or selecting a history entry.
